Things and Inventory

Mining, smelting and crafting produce instances of subtypes of Ore, Alloy, or Crafted, respectively. These are all subtypes of Thing. Each concrete subtype of Thing has a count field indicating a quantity of that Thing.

One can make a Thing using a concrete constructor

using IdlePlanetMinerTools

Battery(3)
Battery(3)

or using the @t_str macro, which makes the named thing with a count of 1.

t"battery"
Battery(1)

Note that for convenience, @t_str uses a dictionary of all concrate Thing subtypes and finds the closest string match.

An Inventory represents a collection of disperate Things.

For convenience, there is a total ordering defined on all concrete subtypes of Thing.

@@ Arithmetic

Arithmetic operations are defined for Things and Inventorys:

using IdlePlanetMinerTools

need = 7 * delta(rx"gravity chamber")

have = GravityChamber(4) + AdvancedComputer(42) + BasicComputer(64) + Circuit(902)

actions, result = crafting_plan(have - need)
(CraftingAction[CraftingAction(Recipie(GravityChamber, Inventory(Thing[AdvancedComputer(-60), NuclearReactor(-1)]), 24300), 3)], Inventory(Thing[Circuit(902), BasicComputer(64), AdvancedComputer(282), NuclearReactor(4), GravityChamber(0)]))
